
# Global Electronics Retail Sales & Customer Insights Analysis

##  Project Overview
This project analyzes global electronics retail data to uncover insights into:
- Sales performance across regions and time
- Customer demographics and buying behavior
- Product and category profitability
- Impact of currency exchange rates
- Store performance and operational efficiency

## 🛠 Tools & Libraries
- Python
- pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook

##  Datasets Used
- Customers: demographic and geographic data
- Exchange Rates: currency conversion data
- Products: product details and categories
- Orders: transactional data
- Stores: physical store information

##  Data Preparation
- Loaded Excel files using `pandas.read_excel()`
- Cleaned missing values and fixed data types
- Merged datasets using:
  - Customer Key
  - Product Key
  - Store Key
  - Currency Code
- Converted all sales values to USD
- Created new metrics:
  - Total Sales
  - Profit
  - Customer Age

##  Key Analyses
- Customer demographics and gender-based revenue
- Global and country-wise sales performance
- Product, brand, and category profitability
- Store performance and size efficiency
- Sales trends over time

##  Visualizations
- Bar charts (sales by country, brand, category)
- Line charts (monthly & yearly trends)
- Heatmaps (price, quantity, profit)
- Scatter plots (store size vs sales)
- Boxplots (age vs spending)
- Pie charts (gender distribution)

##  Business Outcomes
- Identified top-performing regions and products
- Revealed customer segments driving revenue
- Highlighted high-profit categories
- Provided store optimization insights
